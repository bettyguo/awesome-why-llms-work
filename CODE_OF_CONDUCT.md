# Code of Conduct

This is a research-curation repo. We assume good faith and we expect the same in return.

## What we expect

- **Engage with the strongest version of the other person's argument.** Strawmen are uninteresting and we will close threads that rely on them.
- **Cite or concede.** Claims about the literature need a paper; claims about the field need a name. Vibes are not citations.
- **Be specific.** "This is wrong" without a citation is not a contribution.
- **Critique the work, not the person.** Authors of cited papers are not the topic of discussion.

## What is not okay

- Harassment of any kind — on this repo or anywhere reachable from it (linked Twitter accounts, etc.).
- Ad-hominem attacks against contributors or against cited authors.
- Discriminatory language.
- Spamming PRs / issues with auto-generated content.
- Threats, intimidation, or doxing.

## Enforcement

Maintainers may close issues, lock threads, revert PRs, and ban accounts that violate the above. We will explain *why* when we do.

For serious violations or anything that needs handling off the public repo, contact: `conduct@<repo-domain>` (placeholder; update before launch).

## Scope

This applies to all repo spaces — issues, PRs, discussions, and any official chat channels we set up. It does not extend to private discussions among contributors, but if a private interaction would be a violation in public, we will treat the conduct itself as the violation.
